# homebrew-echarge-report

Homebrew tap for [echarge-report](https://github.com/szilch/echarge-report) — a CLI tool for interacting with wallbox charging stations.

## Install

```bash
brew tap szilch/echarge-report
brew install echarge-report
```

## Usage

After installation, run:

```bash
echarge-report --help
```

See the [documentation](https://github.com/szilch/echarge-report/wiki) for configuration and full usage instructions.
